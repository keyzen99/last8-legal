# Privacy Policy (4x8)

Last updated: 2026-03-02

## 1. Information We Collect
The app may process the following data to provide core features:
- Account identifiers (via Firebase Authentication)
- User-uploaded audio files and related metadata (for example, song title and group data)
- Usage analytics (Firebase Analytics)
- Crash and diagnostics data (Firebase Crashlytics)

## 2. Purpose of Use
We use this data to:
- Provide sharing, playback, and chord-analysis features
- Improve quality and stability
- Detect and troubleshoot errors
- Prevent abuse and unauthorized use

## 3. Third-Party Services
4x8 uses Firebase services provided by Google LLC (Authentication / Firestore / Storage / Analytics / Crashlytics).
Data handling by these services is governed by Google's terms and privacy policy.

## 4. Data Retention
Data is retained only as necessary for service operation and may be deleted or anonymized based on operational policy.

## 5. Contact
For privacy-related questions, contact:
- apps@kentaroyamamoto.com

# プライバシーポリシー（4x8）

最終更新日: 2026-03-02

## 1. 取得する情報
本アプリは、機能提供のため以下の情報を取り扱う場合があります。
- 認証情報（Firebase Authentication によるアカウント識別子）
- ユーザーがアップロードした音源ファイルおよび関連メタデータ（曲名、グループ情報など）
- 利用状況データ（Firebase Analytics）
- クラッシュ情報・診断情報（Firebase Crashlytics）

## 2. 利用目的
取得した情報は以下の目的で利用します。
- 共有・再生・コード解析などの機能提供
- 品質および安定性の向上
- 障害検知および不具合対応
- 不正利用対策

## 3. 第三者サービス
本アプリは Google LLC が提供する Firebase（Authentication / Firestore / Storage / Analytics / Crashlytics）を利用しています。
これらサービスのデータ取り扱いは、提供事業者の規約・ポリシーに従います。

## 4. 保存期間
データはサービス運用に必要な期間のみ保持し、運用方針に従って削除または匿名化されます。

## 5. お問い合わせ
プライバシーに関するお問い合わせ先:
